# Quick Start Guide

## Getting Started in 3 Steps

### Step 1: Install
```bash
npm install
```

### Step 2: Start Server
```bash
npm start
```

### Step 3: Access Application
Open browser to: **http://localhost:3000**

## For Students

1. **Register**
   - Click "Register" tab
   - Fill in details (Full Name, Username, Email, Password)
   - Select Role: **Student**
   - Click Register

2. **Enroll in Batch**
   - Login with your credentials
   - Go to "All Batches"
   - Click "Enroll Now" on any batch

3. **Submit Daily Report**
   - Go to "My Daily Report"
   - Select date and batch
   - Fill in your work details
   - Click "Submit Report"

---

## For Teachers

1. **Register**
   - Same as students but select Role: **Teacher**

2. **View Reports**
   - Login with teacher account
   - Go to "Reports" tab
   - View all student reports
   - Use filters to narrow down
